# LuxReplier — AI Business Assistant for Luxembourg SMEs

Multilingual AI assistant that answers customers in French, German, English & Luxembourgish.

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to vercel.com → "Add New Project"
3. Import your GitHub repo
4. Click "Deploy"
5. Done! Your site is live.

## Run locally

```bash
npm install
npm run dev
```

## Built with

- React + Vite
- Claude API (Anthropic)
- Stripe (payments)
